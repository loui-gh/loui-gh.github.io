My first blog entry

Hello world! I am a budding web developer that goes by the name of Loui, and I am using this site to document my journey to becoming a web developer. Here’s where I’m starting from: I have finished 5% of my coding course at 42-Wolfsburg, and I have 3 coursera courses in Python under my belt. In other word, I am a noob, hence the domain name, webdevnoob :) Originally I was just going to slap something together using a theme builder and work my way up from there. I spent 3+ hours this morning figuring out that I needed a domain name and a host (no, I didn’t know those two things were separate), and then another 3 hours deciding on which host was best. I had finally settled on a company in Lithuania when all of a sudden, a friend from 42-Wolfsburg – let’s call him Friendly Guy – came online. Friendly Guy has a background in web development and is a more advanced 42-student. Friendly Guy suggested that I host the website myself. My initial instinct was, “&%!@ no!!”, but I got over my fear of the unknown and instead listened to what turned out to be a bloody good impromptu tutorial on how to set up your own web hosting server.

If you’re interested, here it is in a nutshell:

How to host your own website.

  1. Buy server - I chose Hetzner
  2. Add ssh key to server
    1. $ ssh-keygen
    2. Add id_rsa.pub to Hetzner server
  3. Buy domain name. I got mine for free from namecheap.com thanks to the Github Student Package
  4. Point domain to server --> go to the domain provider, click ‘manage’, ‘Advanced DNS’, remove any other servers there, and add your IP address.
  5. The Hetzner server has nothing on it, so you have to install everything yourself
    • $ sudo apt install nodejs
    • $ sudo apt install npm ←so that you can do a bunch of cool shit later
  6. Add pip, apache and certbot (for SSL, so that your websites have HTTPS). For how-to see: https://www.inmotionhosting.com/support/website/ssl/lets-encrypt-ssl-ubuntu-with-certbot/
    7. Notes: pip is a package manager for Python, npm is a package manager for ; apache2 is open source web server software. Please also note: I don’t really know what any of those things truly mean, I’m just taking a leap of faith for now :) Create config file for your website
  7. Set up your apache config file
    1. Go to apache folder→/etc/apache2/sites-available
    2. Create & modify config file
      3. $ cp 000-default.conf webdevnoob.me.conf
      modify your config file according to the picture below
    3. Activate config file
      4. $ a2ensite webdevnoob.me.conf
    8. PRO TIP ON VIM: to type stuff, press ‘i’ to go to insert mode; to save & quit: ESC+SHIFT+ (little : appears bottom left of screen), enter: wq! + press ‘enter’
  8. Create git repository, clone into /var/www
  9. Once in /var/www, $ git clone [insert repository address here] yourwebsitefolder
  10. Give apache2 rights to yourwebsitefolder
      $ chown -R www-data:www-data /var/www
  11. Set up a package.json file
    • $ npm init
    • [insert image npm_init]
  12. Write your first text
    • Create index.html file in yourwebsitefolder
    • $ vim index.html
    • Write "Hello world!" in your index.html file
